Политика конфиденциальности

Расширение: Traffic.Lab Отчёты

Эта политика описывает, какие данные обрабатывает расширение «Traffic.Lab Отчёты» и в каких целях. Мы соблюдаем требования Магазина Chrome и применяем принцип минимально необходимого доступа.

1. Какие данные мы обрабатываем

Расширение не собирает и не обрабатывает персональные данные пользователей (ФИО, адрес, e‑mail, платежные сведения и т. п.). Для работы с официальными API Яндекса используется OAuth‑токен доступа.

  • Технические данные авторизации: OAuth‑токен для доступа к API Яндекс.Директа и Яндекс.Метрики.
  • Данные страниц браузера, история посещений, содержимое сайтов и действия пользователя не собираются.

2. Источники и получатели данных

Расширение взаимодействует только с официальными доменами Яндекса:

  • https://oauth.yandex.ru/ — получение токена;
  • https://login.yandex.ru/ — информация в рамках OAuth‑потока;
  • https://api.direct.yandex.ru/ — API Яндекс.Директа;
  • https://api-metrika.yandex.net/ — API Яндекс.Метрики.

Данные не передаются третьим лицам и не используются для рекламных целей.

3. Цели обработки

  • Формирование и настройка маркетинговых отчётов на основе данных из API Яндекса.
  • Экспорт и управление данными отчётов по запросу пользователя.

4. Хранение и срок

OAuth‑токен может храниться локально с использованием механизма chrome.storage и/или синхронизации Chrome (если включена пользователем). Срок хранения ограничен сроком действия токена или до его отзыва пользователем. Пользователь может в любой момент выйти из аккаунта/удалить данные расширения в настройках браузера.

5. Безопасность

  • Расширение не загружает и не исполняет удалённые скрипты; политика безопасности контента ограничена значением script-src 'self'.
  • Правила сетевых запросов (например, для CORS) применяются только к перечисленным выше доменам.

6. Права пользователя

Вы можете отозвать доступ в настройках аккаунта Яндекс, удалить локально сохранённые данные расширения через настройки Chrome, а также удалить само расширение из браузера.

7. Дети

Расширение не предназначено для использования детьми младше 13 лет и не обрабатывает данные детей.

8. Изменения политики

Мы можем периодически обновлять эту политику. Актуальная версия всегда доступна по этому адресу. Продолжая использовать расширение после обновления, вы соглашаетесь с новой редакцией.
